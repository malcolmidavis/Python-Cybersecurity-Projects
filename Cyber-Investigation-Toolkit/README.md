# Cyber Investigation Toolkit

## Overview

This Python application provides a small cybersecurity investigation toolkit for collecting user input, organizing investigation details, and supporting basic analyst workflows.

## Features

- Collects investigation-related information
- Organizes cybersecurity case details
- Demonstrates menu-based program structure
- Supports basic investigative documentation

- Conditional Logic
- Menu-Based Programs
