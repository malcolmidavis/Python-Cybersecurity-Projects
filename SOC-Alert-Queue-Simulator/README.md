# SOC Alert Queue Simulator

## Overview

This Python application simulates a Security Operations Center (SOC) alert queue by allowing users to create, prioritize, and review security alerts in a structured workflow.

## Features

- Creates simulated security alerts
- Prioritizes alerts
- Displays alert information
- Demonstrates queue-based processing and analyst workflows

## Skills Demonstrated

- Python
- Lists and Data Structures
- Functions
- Conditional Logic
- User Input Validation
- SOC Alert Management
